# CoffeeShop

## Dependencias
Tenes que tener andando la [coffee-shop-api](https://github.com/Unicen-Tuari/coffee-shop-api)

Pasos
```
git clone https://github.com/Unicen-Tuari/coffee-shop-api.git
cd coffee-shop-api
docker compose up
npm install
npm run start
```
